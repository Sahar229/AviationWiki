# AviationWiki
מדריך התקנה
1) יש להיכנס לשורת הפקודה של תיקיית השורש של הפרויקט
2) נא להריץ את הפקודה הבאה כדי לוודא שהכל הספריות שהמערכת דורשת מותקנות
pip install -r requirements.txt
3) לאחר מכן, נא להריץ גם את הפקודה הבאה בכדי ליצור את קבצי התעודה. יש לקחת את המפתח המודפס על המסך לטובת ההמשך
python utils/keys_generator.py
4) שנו את שם הקובץ “.env.example” ל “.env”
5) פתחו את הקובץ והזינו את הנתונים הבאים
•	GEMINI_API_KEY: המפתח שלכם לשירות  Google GenAI
•	SECRET_KEY: המחרוזת האקראית שהודפסה על ידי סקריפט המפתחות בשלב הקודם (משמשת להצפנה ואבטחת ה-Sessions).
•	CERT_PATH / KEY_PATH: נתיבי תעודות הSSL אם הרצתם את הסקריפט בשלב ב', הנתיבים המוגדרים כברירת מחדל בקובץ (./certs/...) מתאימים ואין צורך לשנותם
•	SENDER_EMAIL: כתובת האימייל שממנה יישלחו הודעות המערכת
•	APP_PASSWORD: סיסמת האפליקציה של חשבון האימייל שלכם (שימו לב: אם אתם משתמשים בGmail אין להזין את הסיסמה הרגילה שלכם. עליכם להיכנס להגדרות חשבון Google שלכם, לחפש "סיסמאות לאפליקציות", וליצור סיסמה ייעודית באורך 16 תווים עבור הפרויקט)
6) לאחר שהכול הושלם, ניתן לחזור לשורת הפקודה ולכתוב 
python run.py
7) כעת יש לגשת לדפדפן ולהקליד בשורת הURL את כתובת הIp המוגדרת והפורט (בתור ברירת מחדל זה יהיה “https://127.0.0.1:5000”)
8) יש לשים לב שבעת הרצה ראשונה טבלת הנתונים הינה ריקה וכדי להתחבר למערכת יש ליצור תחילה משתמש.
